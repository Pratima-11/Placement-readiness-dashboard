# Placement Readiness Dashboard

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

## Installation

### Clone the repository

```bash
git clone https://github.com/Pratima-11/Placement-readiness-dashboard.git
```

### Install dependencies

Backend:

```bash
cd server
npm install
```

Frontend:

```bash
cd client
npm install
```

### Configure MongoDB

Start MongoDB and restore the database if a backup is provided.

### Run the backend

```bash
cd server
npm start
```

### Run the frontend

```bash
cd client
npm run dev
```
